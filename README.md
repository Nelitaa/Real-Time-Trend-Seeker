# 📖 [Real_Time_Trend_Seeker] <a name="about-project"></a>

**[Real Time Trend Seeker]** is a web page for articles where users can find real-time search results and have statistics on what topics people are searching for the most, showcasing trends.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 gem install rails
```

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:Nelitaa/Real-Time-Trend-Seeker.git
```

### Install

Install this project with:

```sh
  cd my-project
  bundle install
```

### Usage

To run the project, execute the following command:

```sh
  rails server
```

### Run tests

To run tests, run the following command:

```sh
  bin/rails test
```
